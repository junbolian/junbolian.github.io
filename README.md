# Junbo Jacob Lian - Academic Homepage

Personal academic homepage for **Junbo Jacob Lian**, M.S. student in Machine Learning & Data Science at Northwestern University.

**Live site:** [https://junbolian.github.io](https://junbolian.github.io)

## Sections

- **Home** - Bio, education, contact, and social links
- **Research** - Research directions and areas of focus
- **Publications** - Papers organized by research area
- **Projects** - Open-source tools, patents, and grants
- **Awards** - Scholarships and competition honors
- **News & Media** - Press coverage and announcements
- **Professional Service** - Journal reviewer roles

## Tech Stack

- Bootstrap 5 + custom CSS
- Markdown content (parsed client-side via [Marked.js](https://github.com/markedjs/marked))
- YAML config (parsed via [js-yaml](https://github.com/nodeca/js-yaml))
- MathJax for LaTeX rendering
- Deployed on GitHub Pages

## Updating Content

Edit Markdown files in `contents/` and push — no build step required.

```
contents/
├── config.yml         # Site title, copyright, etc.
├── home.md            # Bio and contact info
├── research.md        # Research directions
├── publications.md    # Paper listings
├── projects.md        # Open-source projects, patents, grants
├── awards.md          # Scholarships and honors
├── news.md            # Media coverage
└── service.md         # Reviewer roles
```
